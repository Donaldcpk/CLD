# 聖誕節幸運抽獎網頁 (v2.0)

這是一個聖誕節主題的抽獎網頁應用，專為學校設計。

## 功能特點
- **二獎與大獎**：專屬抽獎模式（二獎每級2份，大獎每級1份）。
- **即時顯示**：左側抽獎，右側極大字體顯示中獎名單。
- **名單匯入**：支援 Excel 匯入學生名單。
- **防重複**：自動排除已中獎者。

## 安裝與執行
1. 下載所有檔案 (`index.html`, `script.js`, `styles.css`, 圖片等)。
2. 直接雙擊 `index.html` 開啟即可使用 (建議使用 Chrome 或 Edge 瀏覽器)。

## 使用說明
1. **選擇獎項**：從下拉選單選擇「二獎」或「大獎」。
2. **選擇年級**：選擇要抽獎的年級。
3. **開始抽獎**：
   - **二獎**：點擊後，系統會同時抽出 2 位幸運兒。
   - **大獎**：點擊後，系統會抽出 1 位幸運兒。
4. **補抽模式**：如需補抽，點擊左下角「補抽模式」進行操作。
