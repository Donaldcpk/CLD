# 學校聖誕抽獎系統更新計畫 (V2)

此計畫將進一步優化系統，以符合最新的班級人數數據，調整視覺風格，並加入讀取 Excel 名單以顯示學生姓名的功能。

## 變更目標

1.  **數據校正**：更新所有班級的準確人數。
2.  **視覺優化**：
    -   **配色**：移除紅色字體，改用黃色、橙色或藍色，保持慶典感但避免紅色。
    -   **版面**：調整內容寬度與邊距，確保左右下角的學校吉祥物不會遮擋文字內容。
3.  **姓名顯示**：
    -   允許使用者上傳 Excel (`.xlsx`) 學生名單。
    -   抽獎時若有名單資料，將在大螢幕同步顯示「姓名、班別、學號」。

## 實作細節

### 1. 更新班級數據 (`script.js`)
更新 `this.classData` 為最新數值：
-   **中一**: 1A(27), 1B(28), 1C(29), 1D(29)
-   **中二**: 2A(33), 2B(33), 2C(33), 2D(33)
-   **中三**: 3A(33), 3B(34), 3C(33), 3D(32)
-   **中四**: 4A(26), 4B(30), 4C(23), 4D(23)
-   **中五**: 5A(28), 5B(26), 5C(17), 5D(17)
-   **中六**: 6A(28), 6B(26), 6C(22), 6D(14)

### 2. 介面調整 (`styles.css`)
-   **顏色變數**：
    -   `--accent-red` -> 改為橙色 (`#FF8C00`) 或亮藍色 (`#00BFFF`)。
    -   文字顏色調整，確保在深綠背景上的對比度。
-   **佈局調整**：
    -   `.container` 增加底部 `padding` (約 40vh)，確保滾動到底部時內容不被吉祥物遮擋。
    -   限制 `.container` 的 `max-width` 或增加左右 `padding`，讓內容集中在中間，避開左右兩側的圖像。

### 3. Excel 名單匯入 (`index.html`, `script.js`)
-   **HTML**:
    -   引入 `xlsx.full.min.js` (SheetJS CDN)。
    -   在控制面板加入 `<input type="file">` 供使用者上傳名單表格。
-   **JavaScript**:
    -   監聽檔案上傳事件，解析 Excel 檔案。
    -   建立 `studentMap`: Key=`Class-Number` (e.g., "1A-05"), Value=`Name`.
    -   假設 Excel 格式包含 columns: `Class`, `ClassNo`, `Name` (需具備容錯邏輯或明確指引)。
    -   修改 `drawWinner` 和 `updateWinnersList`，在顯示結果時從 `studentMap` 查找並顯示姓名。
